# Matrix Multiplication

    An application for multiplying 2D matrix of
    matched dimensions.

## Sample input

int[][] A = {{12, 2, 4},
             {10, 1, 1}};

int[][] B = {{2, 2},
             {1, 2},
             {1, 1}};

multiplyMatrix(A, B);
